## Screenshot Guidelines

- **Resolution**: Use 1920x1080 or higher for best quality
- **Format**: Save as PNG for better quality
- **Naming**: Use exact filenames as listed above (lowercase with hyphens)
- **Content**: Ensure no sensitive or real data is visible
- **Browser**: Use Chrome or Edge for consistent rendering
- **Zoom**: Use 100% browser zoom level
